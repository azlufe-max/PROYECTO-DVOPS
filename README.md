# Check-In System (GuestPass)

## Requisitos
* Docker Desktop instalado y encendido.
* Terraform instalado.

## Cómo ejecutar localmente
1. Clona el repositorio.
2. Ve a la carpeta terraform: `cd terraform`
3. Inicializa: `terraform init`
4. Despliega: `terraform apply -auto-approve`
5. Accede al backend en `http://localhost:8000`
